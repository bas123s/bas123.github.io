How To Join

วิธีเข้าเซิร์ฟ

ทำตาม 5 ขั้นตอนนี้แล้วเจอกันใน Knox County

1

เปลี่ยนเป็น Build 42

ใน Steam → คลิกขวา Project Zomboid → Properties → Betas → เลือก unstable

2

เปิด Project Zomboid → กด Multi player

เปิดเกมจาก Steam แล้วกด Multi player จากเมนูหลัก แล้วเพิ่ม Server

3

ใส่ Server Address

ด้านขวาใส่ข้อมูล:
IP: alberto-aerobic.with.playit.plus
Port: 1050
ตั้ง Username + Password ของตัวเอง (จำไว้ ใช้ login ทุกครั้ง)

4

ปิด Steam Relay → Add → เข้าเลย!

☐ ติ๊กออก "Use Steam Relay" ← สำคัญมาก!
กด Add → เลือกเซิร์ฟจากลิสต์ → รอโหลดสักครู่ → เล่นได้!
